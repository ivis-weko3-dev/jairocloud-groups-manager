## bp
リポジトリ管理関連の API を提供する Blueprint インスタンス。

## filter_options
フィルター選択肢の取得を提供するコントローラ。

#### シグネチャ
```python[repositories.py]
@bp.get("/filter-options")
@login_required
@roles_required("system_admin", "repository_admin")
@validate()
def filter_options() -> RepositoryFilterOption:
```

#### エンドポイント
**GET** /api/repositories/filter-options

#### 戻り値

| 型                     | 説明                             |
| ---------------------- | -------------------------------- |
| RepositoryFilterOption | リポジトリのフィルターオプション |

#### 処理内容
1. current_user.is_member_ofでis_member_ofを取得
2. [`service.repositories:get_filter_options`](../05.services/02.repositories.md#get_filter_options)でフィルターオプションを取得
3. RepositoryFilterOptionを返却
